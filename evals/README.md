# Evals

This directory stores manual labels and run-to-run evaluation artifacts.

- `labelled_sets/`: human-labeled records used as reference sets.
- `results/`: eval outputs (metrics, drift summaries, confusion artifacts).

Suggested naming:
- Label sets: `labelled_set_<name>_v1.jsonl`
- Eval results: `eval_<run_id_or_version>.json`
